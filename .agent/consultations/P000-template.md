# ChatGPT Pro Consultation Template

Use this when Mission Control decides a question needs high-reasoning help.

## Prompt to paste into ChatGPT Pro

```text
We are working on PROJECT_NAME.

Context:
- Goal: ...
- Current stage: ...
- Relevant specs/files: ...
- Decision needed: ...
- Constraints: ...

Please analyze the options, recommend one, and produce the output format below.
```

## Expected output format

```markdown
# Consultation Result: TITLE

## Recommendation

...

## Rationale

...

## Spec changes

- file: path
  change: ...

## Task changes

- create/update task: ...

## Risks

...

## Decision record draft

ADR text or bullet points.
```

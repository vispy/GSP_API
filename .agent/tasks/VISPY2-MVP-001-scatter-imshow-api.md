# VISPY2-MVP-001 - Add minimal VisPy2 scatter/imshow producer API

## Goal

Add minimal VisPy2 API that produces GSP point/image scenes and renders via Matplotlib protocol renderer.

## Mission

M008

## Acceptance

- `scatter` produces GSP point visual.
- `imshow` produces GSP image visual.
- Minimal show/render path exists.
- Tests prove output is renderable by Matplotlib protocol renderer.

## Stop conditions

Stop if API design expands toward full Matplotlib compatibility.

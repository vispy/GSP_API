# Development history

This section identifies preserved compatibility and historical material. It is not the recommended
GSP learning path and does not define current protocol semantics.

!!! warning "Legacy renderer API"
    `Canvas`, `Viewport`, `Camera`, legacy visual classes, `RendererBase`, `RendererRegistry`,
    `GSP_RENDERER`, `datoviz-v03`, and the Flask network renderer belong to the older object-oriented
    rendering system.

The repository retains that code, its examples, the former Philosophy essays, implementation stage
records, consultations, closeouts, and QA evidence for maintenance and traceability. They should not
be read as an alternative definition of GSP.

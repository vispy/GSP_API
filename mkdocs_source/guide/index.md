# User guide

GSP currently exposes a broad set of validated protocol records and focused backend reference
paths. These guides will grow around executable examples rather than hypothetical convenience APIs.

| Area | Current scope |
|---|---|
| 2D visuals | Points, markers, segments, paths, images, text, and triangle meshes |
| Views and layout | `View2D`, explicit coordinate spaces, affine transforms, panels, guides, and resolved layout |
| Color | Scalar encodings, linear normalization, canonical named colormaps, and colorbar intent |
| Interaction | Semantic View2D and View3D navigation actions |
| 3D | Orthographic and perspective views, opaque meshes, flat Lambert shading, and bounded picking contracts |
| Data | Buffers, Texture2D records, extension manifests, and bounded data-source contracts |

Start from the [maintained examples](../examples/index.md), then use the
[specification map](../specification/index.md) for exact rules.
